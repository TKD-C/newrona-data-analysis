##과제 : 디스코드 서버 봇 제작 (소규모 서버에서 리그 오브 레전드 내전 경기 기록용도 (5대5 경기))

##언어&프로그램 : c# wpf앱

##기능
1) 플레이어 추가 & 저장 & 수정 (sqlite가 좋으려나?)
- 플레이어 : 이름, 롤 닉네임, 롤 티어, 서버 내 롤 등급 (후술)
2) 5대5 경기 기록을 저장.
- 승패



- 서버 내 롤 등급
OP = 점수 200이상
★★★ = 점수 190이상
★★ = 점수 180이상
★ = 점수 170이상
칼칼칼  = 점수 160이상
칼칼 = 점수 150이상
칼 = 점수 140이상
번번번 = 점수 130이상
번번 = 점수 120이상
번  = 점수 110이상
고양이 = 점수 100이상

(점수 시스템은 나중에 구현하려함)
